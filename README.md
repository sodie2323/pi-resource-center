# pi-resource-center

Pi package that adds a `/resource` command for browsing packages, skills, extensions, prompts, and themes.

## Usage

Install locally:

```bash
pi install E:/code/pi-package
```

Then use:

```bash
/resource
/resource packages
/resource skills
/resource extensions
/resource prompts
/resource themes
/resource sync
```

## Status

Current implementation includes:

- category tabs
- search
- keyboard navigation
- project and user resource discovery
- overlay resource browser
